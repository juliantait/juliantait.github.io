Making oTree data usable

Empirical Analysis Template

A self-contained R + LaTeX template for a single-study empirical paper. The script in Making oTree data usable is one helper inside this larger pipeline.

  • Phases — pre-analysis plan → research plan → data prep → analysis → submission-ready manuscript
  • Scripts/ — config_init, config_cleaning (orchestrator), descriptives, hypotheses, balance_table, robustness, sample_restrictions, exploratory
  • Helper/ — pluggable data-source adapters (otree.R is one of these; also csv.R)
  • LaTeX/ — paper source
  • Context/ — structured documents that brief Claude Code on the project state and per-phase tasks
