Empirical Analysis Template

GitHub

A self-contained R + LaTeX template for a single-study empirical paper. Provides the analysis-pipeline scripts, data-source adapters, paper template, and figure/table style conventions — set up your Data/, run main.R, write up in LaTeX/. Use it manually; no automation required.

Structure

  • Scripts/ — config_init, config_cleaning (orchestrator), descriptives, hypotheses, balance_table, robustness, sample_restrictions, exploratory
  • Helper/ — pluggable data-source adapters (otree.R, csv.R)
  • LaTeX/ — paper source
  • Data/ — raw and cleaned data folders
  • Flow/ — codebook, research_log, timeline, todo, personality (where you record project aim)
  • Tools/ — JEBO-style conventions for publication-ready graphs and tables (skill_graphs.md, skill_tables.md)
  • Literature/ — drop your reading material here

Getting started

Clone the repository to get a working copy:

git clone https://github.com/juliantait/Template_Analysis.git

Claude-orchestrated variant

If you want this template wrapped in a Claude-Code-orchestrated workflow (researcher-personality profiles, phase walks from pre-analysis plan to submission, structured Context/ documents), see the Claude variant: Template_Analysis_Claude on GitHub
